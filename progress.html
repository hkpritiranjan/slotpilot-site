Architecture

The tech stack

Chosen for zero infrastructure cost at launch, with a clear migration path to enterprise scale.

Framework
Next.js 16
App Router, API routes as serverless functions, RSC
Language
TypeScript 5
End-to-end types from DB schema to API response
Database
Turso + Prisma 7
libsql cloud, SQLite-compatible, free tier, no pooling needed
Auth
JWT via jose
Stateless httpOnly cookies, 7-day expiry, no external service
Messaging
Twilio
WhatsApp + SMS from one API. Provider-agnostic abstraction for MSG91, Meta
Styling
Tailwind CSS v4
Utility-first, no design system overhead, fast iterations
Cron
cron-job.org
Free external cron, bypasses Vercel Hobby daily-only limitation
Hosting
Vercel Hobby
GitHub CI/CD, instant deploys, edge network — $0/month
